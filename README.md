# Système de Gestion des Emprunts de Livres — Bibliothèque

Application **desktop** de gestion d'une bibliothèque développée en **Python**, permettant de gérer le catalogue de livres, les adhérents et le suivi des emprunts et retours. Interface graphique réalisée avec **Tkinter**.

> Projet académique — 3ᵉ année Ingénierie Informatique et Réseaux, EMSI Rabat (2025–2026).

## Fonctionnalités

- Gestion du catalogue de livres (ajout, modification, suppression, recherche — CRUD)
- Gestion des adhérents / emprunteurs
- **Suivi des emprunts et des retours** de livres
- Persistance des données en base locale SQLite
- Interface graphique conviviale (Tkinter)

## Technologies

| Outil / Technologie | Rôle dans le projet |
|---------------------|---------------------|
| Python | Langage de programmation |
| Tkinter | Interface graphique (GUI) |
| SQLite | Base de données locale |
| SQL | Requêtes et manipulation des données |

## Installation

```bash
# Cloner le dépôt
git clone https://github.com/h0umm4n3/gestion-bibliotheque-python.git
cd gestion-bibliotheque-python

# Lancer l'application (Tkinter est inclus avec Python)
python main.py
```

> Remplacez `main.py` par le nom réel de votre fichier principal.
